# Roadmap

The milestones are research gates, not promises that every stage is
technically achievable.

1. **Milestone 0: Deterministic observation laboratory.** Build repeatable,
   passive experiments and characterize noise.
2. **Milestone 1: Passive kernel-path discovery.** Begin only after Milestone 0
   acceptance and a written design review.
3. **Milestone 2: Command submission boundary identification.** Identify where
   submissions cross documented and undocumented boundaries.
4. **Milestone 3: Structured command-stream capture.** Investigate safe,
   isolated capture only if earlier evidence supports it.
5. **Milestone 4: Differential field inference.** Test field correlations
   without assigning unsupported semantics.
6. **Milestone 5: Unmodified replay in an isolated environment.** Require a
   dedicated test system and separate safety approval.
7. **Milestone 6: Controlled single-field mutation.** Attempt only after safe,
   deterministic unmodified replay.
8. **Milestone 7: Minimal generated execution profile.** Explore the smallest
   evidence-backed generated workload profile.
